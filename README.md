# TransferPoints

Simple static mock landing page built with vanilla HTML/CSS to demo how an OpenClaw agent could manage transfer-bonus launches for FlightPoints.

## Local preview

```bash
npx serve .
# or
python -m http.server 3000
```

Then open http://localhost:3000.

## Deployment

This repo is published via GitHub Pages from the `gh-pages` branch:
